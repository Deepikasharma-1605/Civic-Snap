CivicSnap is a community-driven platform that enables citizens to report local issues like potholes, broken streetlights, and sanitation problems by simply uploading a photo. Reports are sent directly to the responsible authorities, ensuring faster resolution, transparency, and better civic engagement.

View the app here: https://civic-snap-application.web.app/

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:  
   `npm install`  

2. Set the `API_KEY`  

3. Run the app:  
   `npm run dev`
